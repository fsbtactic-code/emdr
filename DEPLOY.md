# Инструкция по обновлению на сервере Neurocloud

Эта памятка создана для того, чтобы в будущем не искать, где находится папка проекта на сервере `neuroaibolit@neurocloud`.

Всякий раз, когда вы хотите стянуть обновления из GitHub и обновить приложение, просто следуйте этой инструкции:

## 1. Как найти проект, если вы забыли путь
Поскольку проект запущен через `pm2`, вы всегда можете узнать точный путь до него командой:
```bash
pm2 info emdr | grep "script path"
```
Обычно на серверах Ubuntu проекты лежат либо в `~/emdr` (т.е. `/home/neuroaibolit/emdr`), либо в `/var/www/emdr`.

Вы можете также воспользоваться быстрым поиском:
```bash
find ~ -type d -name "emdr" 2>/dev/null
```

## 2. Команды для обновления (Готовый скрипт)

Как только вы перешли в папку (например `cd ~/emdr` или `cd /var/www/emdr`), просто скопируйте и выполните этот блок команд:

```bash
# 1. Скачиваем последние обновления с GitHub 
git pull origin main

# 2. Устанавливаем зависимости (на случай если мы добавляли новые пакеты)
npm run install

# 3. Пересобираем проект Next.js
npm run build

# 4. Перезапускаем PM2 для применения изменений
pm2 restart all
```

> **Совет:** Вы можете объединить это в одну команду, чтобы в будущем просто копировать её:
> `cd /путь/до/папки/emdr && git pull origin main && npm install && npm run build && pm2 restart all`
